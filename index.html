参数 默认值 版本 说明 详细
fixed 0 1.0 (必须)对象fixed时的值:
页面滚动到fixed这个大小时，才fixed，如果不设置fixed的值为对象本身的top
使用: {fixed:150} 		效果
margintop 0 1.0 对象fixed时相对顶部的偏移:
仅当fixed大于对象的top时才有有效，否则fixed大于对象的top时，如果要设置偏离，请调整fixed的大小, top - fixed = 偏移
例如： 需要偏移顶部（50） 对象本身top（150） fixed应设置为 100; 		 
fade 0 1.0 fade效果:
参数: {fade: 0 | 1}
说明： 对象fixed时是否启用fade效果
使用: {fade : 1} 		 
close .btn-close 1.0 关闭按钮:
参数: {close: '.btn-close'}
说明：点击对象内部的按钮，关闭当前对象
使用: {close: '.btn-close'} 		 
css json 1.0 对象本身的css:
说明: 值是一个json对象，和jquery方法 $('xx').css(o) 里面的o一样，需要注意的是当设置bottom时，以bottom为主来定位对象，一般做“返回顶部”按钮使用

事件 参数 版本 说明 详细
fixed (el, st, oh) 1.0 对象fixed时的动作:
说明: el:当前对象，st: window滚动的大小， oh：对象本身的高度
使用: 当对象fixed，设置其背景为红色 		 
unfixed null 1.0 与fixed相反，对象非fixed时的动作:
说明: -
使用: 当对象fixed，设置其背景为黑色

说明: -
使用: 页面上滚动时对象不显示 		 
scrollDown null 1.0 页面下滚动时的动作:
说明: -
使用: 页面下滚动时对象显示
